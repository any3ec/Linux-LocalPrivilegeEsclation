# Operating System(Distribution Type & Version)
Debian Based(like Ubuntu):
```
lsb_release -a
uname -a
cat /etc/os-release
cat /etc/issue
cat /etc/apt/sources.list
cat /etc/debian_version
```
RedHat Based(like CentOs or Fedora):
```
cat /etc/redhat-release
cat /etc/*-release
* ~ put your Disturbution name.
uname -a
```
